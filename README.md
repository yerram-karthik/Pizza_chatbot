# Pizza_chatbot-AWS
For the auto-communicating chat service, I have used Amazon LEX service to build the chatbot and the Kommunicate platform to integrate the chatbot with the pizza ordering website. This chatbot efficiently manages clients and accepts their orders in a simple yet cohesive manner
